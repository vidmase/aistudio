# Product Overview

**Imagina** is an AI-powered image and media editing application that provides comprehensive creative tools through a modern web interface.

## Core Features

- **AI Image Generation**: Text-to-image and image-to-image generation using Gemini and Flux models
- **Image Editing**: Advanced editing with mask-based modifications, object transfer, and generative expand
- **Music Generation**: AI music creation and stem separation using Suno API
- **Video Generation**: AI video creation using VEO models
- **Creative Assistant**: AI-powered suggestions and story-driven creation workflows

## Target Use Cases

- Creative professionals needing AI-assisted image editing
- Content creators requiring music and video generation
- Artists exploring AI-powered creative workflows
- Developers and designers prototyping visual content

## Key Value Propositions

- Unified interface for multiple AI creative tools
- Seamless workflow between different media types
- Professional-grade features with intuitive UX
- Integration with leading AI models (Gemini, Flux, Suno, VEO)
